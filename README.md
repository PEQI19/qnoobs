# qnoobs

A simple implementation of GHZ-based Quantum Anonymous Transmission (https://wiki.veriqloud.fr/index.php?title=GHZ-based_Quantum_Anonymous_Transmission) based on simulaqron for the Pan-European Quantum Internet Hackathon PEQI2019 (https://labs.ripe.net/Members/ulka_athale_1/take-part-in-pan-european-quantum-internet-hackathon).

## Requirements

Requires simulaqron (https://softwarequtech.github.io/SimulaQron/html/index.html) with either projectq or qutip backend.

## Running

```
sh run.sh
```
